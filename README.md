# 地域別家賃比較 Web版

作成日: 2026-07-05
版: v3

## 内容

- `index.html`: GitHub Pages用の静的Webサイト
- `地域別家賃比較_web_20260705_v3_data.csv`: 表示データ
- `地域別家賃比較_web_20260705_v3_target_areas.csv`: SUUMO検索対象

## v3の変更点

- 旧版の「平均家賃」表示を「職場距離」に差し替え。
- 職場距離は、職場所在地から国土地理院住所検索で得た市区町村代表点までの直線距離。
- 郡単位は構成町村代表点の平均点で近似。

## 条件

- 間取り: 2LDK以上
- 面積: 55㎡以上
- 構造: RC/SRC
- 家賃: 賃料＋管理費・共益費込み
- ガス条件: 指定なし
- 非収録: 物価系指標
